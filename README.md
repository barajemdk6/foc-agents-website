# FOC Agents – Static Site

This repository contains a static prototype for FOC Agents with a landing page, authentication pages, and a dashboard.

## Quick Start

- Open `index.html` in your browser, or run a simple local server:

```bash
cd /workspace
python3 -m http.server 8080
```

Then visit `http://localhost:8080/workspace/index.html`.

## Pages

- Landing: `/workspace/index.html`
- Login: `/workspace/login.html`
- Sign Up: `/workspace/signup.html`
- Dashboard: `/workspace/dashboard.html`

## Notes

- The forms are non-functional and redirect to the dashboard for demo purposes.
- Edit styles in `assets/css/style.css` and scripts in `assets/js/main.js`.
